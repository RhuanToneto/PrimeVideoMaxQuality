# Extensão para Máxima Qualidade no Prime Video

PrimeVideoMaxQuality é uma extensão para navegadores baseados em Chromium que força o player do Prime Video a reproduzir automaticamente a melhor qualidade disponível e exibe a resolução detectada no pop-up.

- Funciona em todos os navegadores baseados em Chromium, incluindo:
  - Google Chrome
  - Microsoft Edge
  - Brave
  - Opera

## Instalação

1. Baixe ou clone este repositório.
2. Abra o navegador e acesse a página de extensões:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Opera: `opera://extensions/`
   - Brave: `brave://extensions/`
3. Ative o "Modo do desenvolvedor".
4. Clique em "Carregar sem compactação" e selecione a pasta da extensão.

## Uso

1. Acesse `primevideo.com` e inicie a reprodução de um filme ou uma série.
2. Clique no ícone da extensão na barra do navegador para abrir o pop-up.
3. O pop-up exibirá a resolução detectada.
